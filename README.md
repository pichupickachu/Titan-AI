# Titan AI: High-Frequency Trading Bot

Titan AI is an advanced trading engine that bridges **C++ high-performance processing** with **Gemini 2.0 Flash intelligence**. It's designed to analyze liquidity clusters and market sentiment in real-time.

## Key Features
* **Dual-Language Core:** C++ for low-latency market data; Python for AI reasoning.
* **Sentiment Analysis:** Real-time visual and text analysis via Google Gemini 2.0.
* **Smart Liquidity Detection:** Identifies order book walls and liquidation "magnets."
* **Session Optimization:** Adaptive logic for Asia, London, and New York sessions.

## Tech Stack
* **Core:** C++20 (WebSocket engine)
* **Intelligence:** Python 3.10+, Gemini 2.0 Flash API
* **Data:** orjson (Fast JSON), Pandas, NumPy
* **Visualization:** mplfinance (Real-time charting)

## Repository Structure
* `/src/core` — High-speed C++ engine.
* `/src/analyst` — Python-based AI logic (Titan Brain).
* `/config` — API and risk management configurations.
